## 🚀 Deploy to Vercel (3 Steps)

### Step 1 — Push to GitHub
```bash
# Create a new repo on github.com called "nayrbryan-portfolio"
# Then run:
cd nayrbryan-portfolio
git init
git add .
git commit -m "feat: initial portfolio launch 🚀"
git remote add origin https://github.com/nayrbryanGaming/nayrbryan-portfolio.git
git push -u origin main
```

### Step 2 — Import on Vercel
1. Go to [vercel.com/new](https://vercel.com/new)
2. Click **"Import Git Repository"**
3. Select `nayrbryan-portfolio`
4. Framework: **Next.js** (auto-detected)
5. Click **Deploy** ✅

### Step 3 — Set Custom Domain
1. In Vercel dashboard → Settings → Domains
2. Add: `nayrbryanGaming.vercel.app` (auto-assigned)
3. Or add custom domain if desired
